## Managing Real-Time Data Streams

Building a robust Weather Dashboard is the age-old rite of passage for front-end developers dealing with asynchronous APIs. This project focuses specifically on UI/UX best practices when presenting dense datasets.

### API Integration
Fetching data from public weather endpoints natively returns chaotic JSON blobs. I built custom parsing functions to distill elements down into strictly what the interface required: current temperature, relative humidity, UV index, and an hourly forecast.

### Clean UI
Instead of crowding the screen with numbers, the interface relies on iconography, color theory (warm colors for hot temperatures, cool blues for rain), and spacing to visually communicate the forecast instantaneously.
