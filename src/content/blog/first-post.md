## Welcome to the new blog system

This is pretty amazing. Instead of writing complicated HTML or modifying Javascript arrays just to publish a thought, I can now write plain text documents.

When I save this file into the `src/content/blog` folder, Astro automatically reads the **Frontmatter** at the top of the file (the part between the `---` lines) to figure out the title, date, and excerpt.

### Why this matters:
1. It separates the **content** from the **design**. 
2. It's much faster to write essays or project post-mortems.
3. It enforces a strict schema, so I never accidentally publish a post without a title or date!
